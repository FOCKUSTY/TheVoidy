 # Commit system

- Format: `-SUFFIX NAME.MINI_DESC-FILE`

- Examples:
```
-ref test
-save test
-fix test
-ref-save test.test-test.js
-fix-ref test.desc-ready.listener.ts
-prod.1 is being prepared for production
```

| Suffix  | Description                        |
| ------- | ---------------------------------- |
| --alpha | Alpha version                      |
| --beta  | Beta                               |
| --rel   | Release                            |
| --build | Assembly                           |
| --docs  | Documentation Update               |
| --fix   | Bug fixes                          |
| --hot   | Hot update                         |
| --new   | Adding new functionality           |
| --other | Other                              |
| --prod  | Preparation for release            |
| --ref   | Code refactoring                   |
| --save  | Saving changes                     |
| --sec   | Security                           |
| --style | Code style update                  |
| --test  | Testing, adding test               |
| --ver   | Version update                     |
| --log   | Updated logging                    |
